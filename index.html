Project on console

Visual Studio Code is a source-code editor, which has been made by Microsoft and is available for Windows,Linux and macOS.

The console application is a program in Visual Studio, which accepts the input parameter, calls what service is required and sends the output to the console. When on the live server, to open the consle screen, you need to do a right click and you need to click on the option inspect. To make sure something goes onto the console, use this code: console.log().
